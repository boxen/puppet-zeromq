# 0MQ Puppet Module for Boxen

[![Build Status](https://travis-ci.org/boxen/puppet-zeromq.png?branch=master)](https://travis-ci.org/boxen/puppet-zeromq)

## Usage

```puppet
include zeromq
```

## Required Puppet Modules

* boxen
* homebrew
* stdlib

## Developing

Write code.

Run `script/cibuild`.
